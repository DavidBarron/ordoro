# ordoro-interview
Python version: Python 3.7.2

### set up virtiual env
```
python3 -m venv venv
. venv/bin/activate
pip3 install -r requirements.txt
```

### running script
```
python3 david_barron_ordoro.py
```

### running tests
```
python3 ordoro_test.py 
```

### deactivate virtual env
```
deactivate
```
